# RemoteDesk

RemoteDesk is a lightweight Telegram bot that lets you remotely control your **keyboard** and **mouse**, as well as capture screenshots when needed.  

## Features
- Instant screenshots
- OCR text extraction
- Simple Telegram interface
- Minimal setup, works out of the box

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Romashkaa/RemoteDesk.git
cd RemoteDesk
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file:

```bash
TOKEN=YOUR_TELEGRAM_BOT_TOKEN
ADMIN=YOUR_TELEGRAM_USER_ID

SCREENSHOT_PATH=media/screenshot.png
```

---

## Run the bot

```bash
python main.py
```
